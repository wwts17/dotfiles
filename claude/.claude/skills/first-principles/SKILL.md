# 第一性原理

## 核心
不接受惯例,把结论拆到不可再问的事实,再从底层重建。

## 适用场景
某个"一向如此"的假设挡住了真问题;或要对比差异极大的方案。
不用:问题已解决 / 现有方案够用 / 时间紧。

## 思考步骤
1. 写下你怀疑的结论
2. 列出它依赖的假设
3. 逐个追问每个假设"为什么",直到触底(物理/数学/契约事实)
4. 丢掉站不住的,从事实往上重建

## 输出结构
- 被质疑结论:电池 $600/kWh"因为贵"
- 触底事实:原料成本 ≈ $80/kWh
- 重建结论:$520 是工艺/规模问题,非物理极限——攻这里
