# 四步分析

## 核心
把四个思维工具按序串起来,每步喂给下一步:MECE → 第一性原理 → 系统思考 → 概率思考。

## 适用场景
问题既新颖又高风险、单个工具不够时。
不用:熟悉的问题,单个工具就够。

## 思考步骤
1. 用 MECE 把问题拆成不重不漏的子问题
2. 用第一性原理追到每个子问题的触底事实,丢掉站不住的假设
3. 用系统思考把事实映射成存量/流量/反馈环,找杠杆点
4. 用概率思考给杠杆点估期望值,决定押哪个,设好止损线

跳步可以,但要写明跳了哪步、为什么;各步细节见 `mece-decomposition`、`first-principles`、`systems-thinking`、`probabilistic-thinking`。

## 输出结构
- 子问题(MECE):A / B / C
- 触底约束(第一性原理):每个子问题剩下的硬约束
- 杠杆点(系统思考):改哪个环路撬动最大
- 押注(概率思考,按 EV 排序):押 X,止损线 Y
